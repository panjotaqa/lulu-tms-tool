Como um visitante do site,
Eu quero preencher um formulário com meus dados básicos (nome, e-mail e senha),
Para que eu possa ter um perfil para acessar o sistema

1. A rota deve ser pública permitindo quaisquer usuário de realizar acesso e efetuar o cadastro na plataforma
